# Basic HTML Website

This project is part of the **Roadmap.sh Basic HTML Website** challenge.  
It demonstrates the use of **pure HTML** to build a structured, semantic, and accessible web page without using CSS or JavaScript.

## Project Page
https://roadmap.sh/projects/basic-html-website

## Live Demo
https://78kabita.github.io/basic-html-website/

## Features
- HTML-only website (no CSS or JavaScript)
- Semantic HTML elements (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
- Proper heading hierarchy
- Navigation menu
- Projects, articles, education, and reviews sections
- SEO-friendly meta tags

## Technologies Used
- HTML5

## What I Learned
- Writing clean and semantic HTML
- Structuring a webpage using meaningful tags
- Improving accessibility and readability
- Understanding basic SEO concepts with meta tags
